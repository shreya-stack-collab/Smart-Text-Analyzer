Smart-Text-Analyzer

   A sleek, professional web-based utility designed for real-time text analysis and
   formatting. Built as part of the Digital Heroes project, this tool helps writers
   and editors optimize their content with instant insights.

🚀 Overview

   The Smart Text Analyzer Pro is a lightweight "all-in-one" text workstation. It
   provides a distraction-free environment to draft content while receiving live
   feedback on metrics, readability, and keywords.

✨ Key Features

  - Live Metrics: Instant tracking of word count, character count, and sentence
    count as you type.
  - Time Estimates: Calculates estimated Reading Time and Speaking Time based on
    industry-standard speeds.
  - Keyword Extraction: Automatically identifies the top recurring keywords to
    help with SEO and content focus.
  - Formatting Tools: One-click conversion for UPPERCASE, lowercase, and Title
    Case.
  - Smart Auto-Save: Automatically saves your progress to your browser’s local
    storage so you never lose your work on refresh.
  - Quick Actions: Integrated "Copy to Clipboard" and "Clear All" functions for
    a faster workflow.

🛠️ How It Works

  1.  Input: Type or paste your text into the main editor.
  2.  Analyze: The dashboard updates instantly with statistics and keyword
      density.
  3.  Format: Use the tool buttons to adjust the casing of your text.
  4.  Export: Click "Copy Text" to take your processed content anywhere.

🔒 Privacy & Tech

  - Privacy: All text processing is done locally in your browser. No data is
    sent to a server.
  - Technology: Built using modern HTML5, CSS3 (Inter UI font), and Vanilla
    JavaScript.
